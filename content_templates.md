# Introduction

The Atomic-Learning organisation contains a series of repositories which are designed to act as templates for content pages. Most pages can be created from the basic template, but other templates are designed to make it easier to create content pages with specific features, e.g. a content page which supports a Codespace with Python and Jupyter Notebooks. 

In the case of templates which support Codespaces, they will often using an [image](images.md) as a base.

# Usage

To create a new content page from a template you can either use the "Use this template" button on the relevant repository or, when creating a new repository you can select the template from the "Template" dropdown menu. This will create a new repository with the same files as the template, which you can then edit to create your content page.
